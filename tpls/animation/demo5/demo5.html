Social Icons DisplayUsing CSS3

Marcofolio.net

Simply hover over the social media icons below the see the beautiful effect. The first example is created using pure CSS, and can only be fully viewed using -webkit based browsers (like Chrome and Safari) and Firefox 4. Animations are done using CSS.

The second example uses the same HTML, some of the CSS from the first example but uses jQuery for the animations. This is added for all non-webkit browsers (like older versions of Firefox and Opera).

The idea is an improved way on how Pubwich displays his social media icons. The icons used are created by Paul Robert Lloyd.

More script and css style : www.htmldrive.net
